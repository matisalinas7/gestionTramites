Bienvenido al sistema de Gestión de Trámites

Aplicación web para el registro, seguimiento y administración de trámites empresariales.

Este sistema permite a los usuarios ingresar trámites, cargar documentación, y monitorear su estado actual. Los consultores asignados podrán gestionar el flujo del trámite conforme a los circuitos definidos para cada tipo.

Proyecto académico desarrollado en grupo para la materia Diseño de Sistemas - 3er año.
